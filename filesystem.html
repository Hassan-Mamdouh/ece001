File System

A file is a collection of related information named for secondary storage, such as magnetic disks, magnetic tapes, and optical disks. A file is generally a sequence of bits, bytes, lines, or records whose meaning is defined by the creator and user of the files.

File Structure

A File Structure should be in accordance with the required format that the OS can understand.

  • A file has a certain structure which is defined by its type.
  • A text file is a sequence of lines organized into characters.
  • A source file is a sequence of functions and procedures.
  • An object file is a sequence of bytes which are organized into blocks which the machine may understand.
  • As different file structures are defined by the operating system, it also contains the code to support the file structure. Unix, MS-DOS support minimum file structure number.
