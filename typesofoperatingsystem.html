Types of Operating System

There are many types of Operating System, but we will discuss in some details three of them, which are; Network operating system, time-sharing operating system and Distributed operating system.

Type Definition Advantages
Network operating System It runs on a server and provides the server the capability to control data, users, groups, security, applications and other networking functions.
  • Centralized servers are highly stable.
  • Security is managed on computer.
  • NewÂ technologies and hardware upgrades can be easily integrated into the system.
  • Different locations and types of systems allow remote access to servers.
    		•
    Time-sharing operating systems Time-sharing is a technique that allows several users to jointly use a particular computer system, located at various terminals.
  • Provides the quick response advantage.
  • Avoids software duplication.
  • Decreases CPU idle time.
    		•
    Distributed operating System Distributed systems use several central processors to serve various applications and multiple users in real time. Data processing jobs are also distributed among the processors.
  • Through the facility of resource sharing, an user at one site can be able to use the available resources at another.
  • Speed up data exchange via electronic mail with one another.
  • In a distributed system, if one site fails, the remaining sites can potentially continue operating.
  • Better service to the customers.
  • Load reduction on host computer.
  • Reduction of the data processing delays.
